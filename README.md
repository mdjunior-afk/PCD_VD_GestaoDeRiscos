# Data Warehouse para Gestão de Riscos
Este projeto acadêmico tem como objetivo o desenvolvimento de um Data Warehouse para a Appian Capital, fundo de investimento especializado no setor de mineração, voltado à gestão e visualização de riscos operacionais de seus ativos. A solução integra dados provenientes de diferentes fontes e processos internos, consolidando-os em um repositório centralizado que alimenta um dashboard no Power BI — permitindo uma visualização clara, estruturada e acessível da matriz de riscos para a equipe e seus respectivos responsáveis.
